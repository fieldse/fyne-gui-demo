## Fyne GUI widget

A simple GUI widget for testing Fyne on Mac M1 silicon

### Requirements

-   Macbook with M1 silicon hardware
-   Go 1.19+ (may work with older versions)
-   System libraries? (maybe need to install something via `brew`)

### Installation

Build the project

```sh
go mod tidy
go build app/main.go

```

### Usage

```sh
go run app/main.go
```
